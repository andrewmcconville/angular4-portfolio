Hi, I’m Andrew McConville; I work with product & design on creating user workflows, wireframes and adhering to user interface consistency. I also document reusable components, maintain pattern libraries and build user interfaces. I believe in user centered design, style guides and grid systems.

Work Examples

Looking to dive right in and see my work? Here are a few examples to get you started.

Stack Overflow User Research

I started conducting user research with Stack Overflow in 2017 for my Graduate Certificate in User Experience at the University of Wisconsin-Madison. The project involved feedback and mentorship from Stack Overflow’s product and user research team.

Mesmer

Mesmer is a Trello like degree planner I designed and prototyped in 2012 that allows college courses to be easily reorganized via drag and drop.

DK Handbook

The DK Handbook project prototypes interactions from several chapters of the printed book. It's a translation from print to tablet that features audio, video, drag & drop, and games.
